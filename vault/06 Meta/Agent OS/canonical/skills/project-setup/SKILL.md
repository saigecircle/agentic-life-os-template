# Project Setup Skill

## When to use

Use when creating a new time-bound project.

## Procedure

1. Create a folder under `01 Projects/`.
2. Add `README.md` with goal, status, next action, and file map.
3. Add `Log.md` with the first dated entry.
4. Keep the structure simple.

## Verification

The project has a clear definition of done and a next action.

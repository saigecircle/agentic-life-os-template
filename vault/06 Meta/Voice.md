# Voice

Describe your writing and communication preferences here.

Use this when an agent is drafting emails, posts, docs, proposals, or anything meant for another human.

## I like

- Clear, plain language.
- Useful structure.
- Direct recommendations when there is a real trade-off.
- Warmth without filler.

## I dislike

- Overcomplication.
- Vague summaries.
- Generic motivational language.
- Pretending certainty when the evidence is thin.

## Default tone

<!-- Replace this with your own tone guidance. -->

- Plain English.
- Calm and practical.
- Friendly but not performative.
- Specific enough to be useful.

## Editing rules

- Preserve meaning before style.
- Cut filler.
- Prefer concrete verbs.
- Keep jargon only when the audience already uses it.
- Ask before changing intent, claims, or commitments.

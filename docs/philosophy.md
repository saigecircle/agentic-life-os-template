# Philosophy

A Life OS is not a place to hoard notes.

It is a durable context layer for your life, work, projects, relationships, responsibilities, and reusable workflows.

The system is designed around three principles:

1. **Tool-agnostic** — your context should not be trapped inside one AI app, model, or agent harness.
2. **Human-readable first** — you should be able to open the files and understand what is happening.
3. **AI-usable** — agents should be able to find the right context, follow local rules, and help safely.

The vault is home. AI tools are clients.

# First 15 Minutes

This guide gets you from empty template to useful vault.

## 1. Install the vault

Use the installer from the main README, or copy the contents of `vault/` into a folder you choose.

## 2. Open it

Use Obsidian, VS Code, Cursor, Notepad, or any Markdown editor.

## 3. Read the vault README

Open the installed vault and read:

```text
README.md
```

That file is the shared map for you and any AI agent you invite into the folder.

## 4. Fill in one personal context file

Open:

```text
06 Meta/About Me.md
```

Add only what you are comfortable storing locally. Start small: role, current priorities, preferred working style, and anything an assistant should know before helping.

## 5. Create one project

Create a folder under:

```text
01 Projects/
```

Use the starter project already inside the vault as your model:

```text
01 Projects/Example Project/
├── README.md
└── Log.md
```

Copy that shape for your first real project. A good first project has a clear finish line and one next action.

## 6. Add one ongoing area

Create or update one area under:

```text
03 Areas/
```

Areas are responsibilities that continue over time: health, finances, business operations, learning, home, or your own Life OS.

## 7. Capture one messy thing

Put a note, link, idea, or rough transcript into:

```text
00 Inbox/
```

Do not overthink where it belongs yet. The Inbox exists so capture stays easy.

## 8. Try one AI prompt

Ask your AI assistant:

```text
Read README.md first. Then help me decide where the note in 00 Inbox belongs. Explain your reasoning before moving anything.
```

## Done

You now have the basic loop:

```text
capture → clarify → organise → continue later
```

The system becomes useful because humans and agents share the same map of what matters.

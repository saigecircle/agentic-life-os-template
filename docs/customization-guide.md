# Customization Guide

Start simple.

The purpose of the starter vault is to give you a shared map for yourself and your agents. Do not redesign everything on day one.

Recommended first changes:

1. Fill in `06 Meta/About Me.md`.
2. Edit `06 Meta/Voice.md`.
3. Create one real project.
4. Add one real area.
5. Capture one messy inbox item.
6. Ask an AI agent to help file that inbox item.

Only rename folders once you understand what role each folder plays.
